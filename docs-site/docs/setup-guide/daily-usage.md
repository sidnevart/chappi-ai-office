# Ежедневное использование

## Основное взаимодействие

Всё общение происходит через Telegram. Просто пишите боту как обычному человеку.

### Типичные запросы

**Задачи и память:**
```
Запомни: встреча с командой каждую пятницу в 15:00
Создай задачу: подготовить презентацию до конца недели
Покажи мои текущие задачи
Что ты обо мне знаешь?
```

**Исследования:**
```
Исследуй лучшие практики разработки API в 2026 году
Найди и сохрани информацию о pgvector — что это и как использовать
Сравни подходы к мониторингу микросервисов
```

**Работа с файлами и сервисами:**
```
Создай Google Документ: "Архитектура проекта"
Добавь в таблицу "Расходы" строку: сервер VPS — 20 евро/месяц
Проверь мои последние письма в Gmail
```

**Системные задачи:**
```
Проверь состояние всех сервисов на сервере
Покажи использование диска
Перезапусти Star Office UI
```

Просто запишите голосовое сообщение в Telegram — агент расшифрует и обработает автоматически.

## SDLC и approval через Telegram

```
Покажи статус задач в проекте
Одобри спеку spec-pvti_42
Поставь задачу #42 на паузу
```

## Сердцебиение

Несколько раз в день агент самостоятельно проверяет почту и календарь. Если есть что-то важное — пришлёт сообщение без вашего запроса.

## Советы

**Будьте конкретны** — чем точнее запрос, тем лучше результат:
```
❌ Напиши что-нибудь про Python
✅ Напиши краткое сравнение FastAPI и Django REST Framework для небольшого API
```

**Указывайте модель** для важных задач:
```
Используй тяжёлую модель: проанализируй этот архитектурный документ...
```

**Уточняйте формат:**
```
Ответь списком, не более 5 пунктов
Дай развёрнутый технический ответ
Объясни как будто я не знаком с этой технологией
```

## Управление через Claude Code

Для более глубокого управления AI Office запустите Claude Code в папке проекта:

```bash
cd ~/Documents/Projects/ai_office
claude
```

Там доступны специализированные агенты для инфраструктуры, исследований и базы знаний.
