# Уведомления

Агент отправляет уведомления в Telegram об успешных и неудачных операциях.

## Типы уведомлений

| Тип | Когда приходит | Пример |
|-----|----------------|--------|
| ✅ Успех | Задача завершена | «Исследование по CI/CD готово» |
| ❌ Ошибка | Что-то пошло не так | «Не удалось подключиться к GitHub» |
| ℹ️ Информация | Важное событие | «Агент перезапущен» |
| ⚠️ Предупреждение | Требует внимания | «Диск заполнен на 85%» |

## Как это работает

После каждой значимой операции агент вызывает скрипт уведомлений:

```bash
python3 /opt/ai-office/notify/notify.py success "Задача выполнена: ..."
python3 /opt/ai-office/notify/notify.py failure "Ошибка при ..."
```

Сообщение приходит напрямую вам в Telegram от того же бота.

## Настройка

Уведомления настраиваются в файле `/root/.openclaw/workspace/AGENTS.md` на сервере. Агент знает когда и какие уведомления отправлять.

Для ручной проверки работоспособности:

```bash
ssh root@80.74.25.43
python3 /opt/ai-office/notify/notify.py info "Тестовое уведомление"
```

## Отключение уведомлений

Если уведомлений слишком много, можно попросить агента:

```
Не присылай уведомления о мелких задачах
Отправляй уведомления только об ошибках
```

Агент обновит своё поведение и сохранит предпочтение в памяти.
