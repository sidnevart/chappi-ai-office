# Дашборды и наблюдаемость

## Star Office UI

**Адрес:** [http://80.74.25.43:3000](http://80.74.25.43:3000)

Пиксельный визуальный дашборд — интерактивное представление вашего ИИ-офиса. Каждый активный агент отображается как отдельный персонаж в офисе.

### Что отображается

- **Состояние агентов** — что делает каждый агент прямо сейчас
- **Текущая задача** — краткое описание активной операции
- **Статус** — `idle` (ожидание), `writing` (пишет), `researching` (исследует), `executing` (выполняет), `syncing` (синхронизирует), `error` (ошибка)

### Дополнительные эндпоинты

| Адрес | Назначение |
|-------|-----------|
| `/agents` | Список всех зарегистрированных агентов (JSON) |
| `/token-stats` | Статистика использования токенов и стоимость |
| `/task-history` | История задач и исследовательских отчётов |

### Несколько агентов в офисе

По умолчанию в офисе виден один персонаж — основной агент OpenClaw. Когда запускаются специализированные субагенты (исследователь, инфраструктурный агент, агент базы знаний), они регистрируются в Star Office UI как отдельные персонажи и появляются в офисе.

## Grafana

**Адрес:** [http://80.74.25.43:4000](http://80.74.25.43:4000)  
**Логин:** `admin` / `aio-grafana-2026`

Профессиональный дашборд метрик на основе данных из PostgreSQL.

### Дашборд «AI Office Observability»

Основной дашборд включает следующие панели:

| Панель | Что показывает |
|--------|---------------|
| Активность агента (последние 24 часа) | Количество действий по часам |
| Топ инструментов | Какие инструменты агент использует чаще всего |
| Статус последней сессии | Последнее действие агента |
| Стоимость по моделям (неделя) | Расходы на каждую модель в долларах |
| Токены (сегодня) | Входящие и исходящие токены за день |

### Источник данных

Grafana читает данные из таблицы `event_log` в PostgreSQL. Каждое значимое действие агента записывается туда автоматически.

### Структура таблицы event_log

```sql
SELECT agent_id, state, tool_name, model,
       tokens_in, tokens_out, cost_usd,
       task_summary, created_at
FROM event_log
ORDER BY created_at DESC
LIMIT 10;
```

## Просмотр базы знаний

Для прямого просмотра данных через командную строку:

```bash
# Последние 5 заметок
PGPASSWORD=postgres psql -h 80.74.25.43 -p 5432 -U postgres -d ai_office \
  -c "SELECT content, created_at FROM notes ORDER BY created_at DESC LIMIT 5;"

# Активные задачи
PGPASSWORD=postgres psql -h 80.74.25.43 -p 5432 -U postgres -d ai_office \
  -c "SELECT title, priority, status FROM tasks WHERE status != 'done';"
```
