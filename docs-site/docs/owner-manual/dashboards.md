# Дашборды и наблюдаемость

## Два интерфейса — выбирай свой

У AI Office два визуальных интерфейса. Они не конкурируют — каждый подходит для разных задач.

| | Star Office UI | OpenClaw Office UI |
|---|---|---|
| **Адрес** | [http://80.74.25.43:3000](http://80.74.25.43:3000) | [http://80.74.25.43:3001](http://80.74.25.43:3001) |
| **Стиль** | Пиксельный, уютный офис | Изометрический BigTech |
| **Стек** | Flask + Phaser.js | React + Vite + SVG-анимации |
| **Подключение** | REST API (/set_state) | WebSocket (ws://localhost:18789) |
| **Когда использовать** | Наблюдать за состоянием агентов | Живое взаимодействие с OpenClaw |

---

## Star Office UI

**Адрес:** [http://80.74.25.43:3000](http://80.74.25.43:3000)

Пиксельный визуальный дашборд — интерактивное представление вашего ИИ-офиса. Каждый активный агент отображается как отдельный персонаж в офисе.

### Что отображается

- **Состояние агентов** — что делает каждый агент прямо сейчас
- **Текущая задача** — краткое описание активной операции
- **Статус** — `idle`, `writing`, `researching`, `executing`, `syncing`, `error`

### Дополнительные эндпоинты

| Адрес | Назначение |
|-------|-----------|
| `/agents` | Список всех зарегистрированных агентов (JSON) |
| `/token-stats` | Статистика использования токенов и стоимость |
| `/task-history` | История задач и исследовательских отчётов |

### Несколько агентов в офисе

Когда запускаются специализированные субагенты (исследователь, инфраструктурный агент, агент базы знаний), они регистрируются как отдельные персонажи и появляются в офисе.

## OpenClaw Office UI

**Адрес:** [http://80.74.25.43:3001](http://80.74.25.43:3001)

Современный изометрический интерфейс в стиле BigTech. Подключается к OpenClaw напрямую по WebSocket — показывает живое состояние агента и поток мыслей в реальном времени.

### Особенности

- **Живое подключение** — WebSocket к OpenClaw gateway (ws://localhost:18789)
- **Поток мыслей** — видно что агент «думает» в процессе работы
- **SVG-анимации** — красивые изометрические переходы состояний
- **BigTech атмосфера** — серьёзный корпоративный стиль

### Когда использовать

Используй когда хочешь наблюдать за активной сессией агента в реальном времени или показать кому-то AI Office в действии.

---

## Grafana

**Адрес:** [http://80.74.25.43:4000](http://80.74.25.43:4000)

Профессиональный дашборд метрик на основе данных из PostgreSQL.

### Дашборд «AI Office Observability»

| Панель | Что показывает |
|--------|---------------|
| Активность агента (последние 24 часа) | Количество действий по часам |
| Топ инструментов | Какие инструменты агент использует чаще всего |
| Статус последней сессии | Последнее действие агента |
| Стоимость по моделям (неделя) | Расходы на каждую модель в долларах |
| Токены (сегодня) | Входящие и исходящие токены за день |

Grafana читает данные из таблицы `event_log` в PostgreSQL.

## Просмотр базы знаний

```bash
# Последние 5 заметок

  -c "SELECT content, created_at FROM notes ORDER BY created_at DESC LIMIT 5;"

PGPASSWORD=postgres psql -h 80.74.25.43 -p 5432 -U postgres -d ai_office \
  -c "SELECT title, priority, status FROM tasks WHERE status != 'done';"
```
