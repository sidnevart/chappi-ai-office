# Chappi AI Office

**Chappi AI Office** — это персональный искусственный интеллект-помощник, который работает круглосуточно на вашем сервере и общается с вами через Telegram.

## Что это такое

Chappi AI Office — это не просто чат-бот. Это полноценный рабочий кабинет на основе ИИ:

- **Всегда на связи** — работает на виртуальном сервере, не зависит от вашего компьютера
- **Личная база знаний** — запоминает факты, задачи, ссылки и исследовательские отчёты
- **Голосовые сообщения** — расшифровывает голос и обрабатывает как текст
- **Интеграция с сервисами** — Google Документы, Google Таблицы, GitHub, Notion, Jira
- **Мультиагентная работа** — может параллельно запускать специализированных агентов
- **Наблюдаемость** — дашборды в реальном времени через Star Office UI и Grafana

## Компоненты системы

| Компонент | Назначение | Адрес |
|-----------|------------|-------|
| OpenClaw | Ядро агента, обработка Telegram | — |
| Star Office UI | Визуальный дашборд агента | `:3000` |
| Grafana | Метрики и аналитика | `:4000` |
| PostgreSQL | База знаний | `:5432` |
| Ollama | Языковые модели | `:11434` |

## Как это работает

```
Вы → Telegram → OpenClaw (VPS) → модель ИИ → ответ
                    ↓
               PostgreSQL (база знаний)
               MemPalace (семантическая память)
               Composio (внешние сервисы)
```

1. Вы отправляете сообщение в Telegram боту `@sidnevart_space_bot`
2. OpenClaw на сервере получает сообщение и обрабатывает его
3. При необходимости агент обращается к базе знаний, запускает инструменты или вызывает внешние сервисы
4. Ответ возвращается в Telegram

## Уровни моделей

Система автоматически выбирает подходящую модель по сложности задачи:

| Уровень | Модель | Когда используется |
|---------|--------|-------------------|
| Лёгкий | qwen3:8b | Простые вопросы, краткие ответы |
| Средний | glm-5:cloud | Анализ, рабочие задачи |
| Тяжёлый | kimi-k2.5:cloud | Глубокое исследование, сложные рассуждения |
