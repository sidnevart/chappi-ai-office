# Голосовые сообщения

Вы можете общаться с агентом голосом через Telegram — голосовые сообщения автоматически расшифровываются.

## Как это работает

1. Запишите голосовое сообщение в Telegram
2. OpenClaw получает файл и передаёт его на расшифровку
3. Faster-Whisper (локальная модель на сервере) преобразует речь в текст
4. Агент обрабатывает текст как обычное сообщение
5. Ответ приходит текстом с подтверждением: «🎙️ Понял: "ваши слова"»

## Характеристики

- **Модель расшифровки:** Whisper base (локальная, без внешних сервисов)
- **Язык:** автоматическое определение (русский и английский работают хорошо)
- **Задержка:** 3–10 секунд в зависимости от длины сообщения
- **Ограничения:** очень длинные голосовые (>5 минут) могут расшифровываться долго

## Технические детали (для обслуживания)

Скрипт расшифровки находится на сервере:

```bash
/opt/ai-office/voice/transcribe.py
```

Запуск вручную для тестирования:

```bash
ssh root@80.74.25.43
python3 /opt/ai-office/voice/transcribe.py /tmp/test-voice.ogg
```

Смена модели (если нужна лучшая точность):

```bash
# Варианты: tiny, base, small, medium, large-v3
# Чем крупнее модель, тем точнее, но медленнее
WHISPER_MODEL=small python3 /opt/ai-office/voice/transcribe.py audio.ogg
```

## Текстовые ответы

Агент отвечает только текстом. Голосовые ответы не используются — это осознанное решение для экономии ресурсов и ясности общения.
