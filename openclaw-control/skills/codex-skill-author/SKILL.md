# Codex Skill Author

- Keep triggers concrete and mildly pushy so the skill activates reliably.
- Move long usage examples into references.
- Define eval prompts for the intended role and workflow.
- Avoid bundling unrelated responsibilities into one skill.
- Reference: `docs/skills-strategy.md`
