# OpenClaw Skill Registry Manager

- Keep foundation skills scoped by role, never globally sprayed.
- Prefer many small custom skills over one giant prompt.
- Move long guidance into references and keep triggers explicit.
- Track ownership, role pack, and approval surface for each skill.
- Reference: `docs/skills-strategy.md`
